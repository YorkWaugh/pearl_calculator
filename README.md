# Pearl Cannon Calculator

A lightweight, single-file web tool designed to calculate optimal TNT configurations and simulate flight trajectories for **pre-aligned (zero-offset) Ender Pearl cannons** in Minecraft (1.21.2+).

## Features

- **Precise Calculation:** Solves for exact Blue/Red TNT amounts to reach target coordinates.
- **Trajectory Simulation:** Visualizes the pearl's flight path tick-by-tick.
